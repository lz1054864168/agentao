# Agentao - Interactive Demo

This guide walks you through the key features of Agentao with real examples.

## Setup (30 seconds)

```bash
# 1. Configure API key
cp .env.example .env
# Edit .env and add your OPENAI_API_KEY

# 2. Install dependencies
uv sync

# 3. Run Agentao
./run.sh
```
